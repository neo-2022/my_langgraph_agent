# Правила работы Codex (обязательно)

1. Перед завершением каждого этапа проверять активный PR на GitHub:
   - комментарии ревью;
   - статусы всех проверок (checks);
   - блокеры merge.
2. Если PR открыт по текущей задаче — доводить до закрытия:
   - исправить замечания;
   - добиться завершения всех проверок;
   - обновить комментарий в PR с итогом.
3. Не переходить к следующему этапу, пока текущий PR не доведён до завершённого состояния.
